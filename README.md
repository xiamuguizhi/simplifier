# simplifier
simplifier-typecho主题（极简）

## 简介

极简主题，自适应 无多余css js

适合纯写作，记录生活博友

## 提示

使用前 请到/usr/uploads/新建文件夹avatar 并设置777权限！（头像缓存功能）

网站底部联系邮箱，请到主题设置界面添加
